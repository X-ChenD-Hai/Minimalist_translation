## 使用说明
- ### 点击左上角程序图标右侧菜单可可选择功能
    - ### 窗口置顶可使应用停留在所有窗口顶端
    - ### 字体大小调整可以调整翻译窗口的字体大小
    - ### 剪贴板跟踪可以在复制文本到系统剪贴板时自动粘贴到翻译窗口
